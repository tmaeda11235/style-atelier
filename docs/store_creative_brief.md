# Store Creative Brief & ASO Strategy

This document outlines the layout concepts, benefit-driven messaging, and localization assets for the app store listing of Style Atelier.

## 1. App Store Title & Keyword Targets

- **English**: `Style Atelier - Midjourney Prompt Generator & Organizer`
- **Japanese**: `Style Atelier - Midjourney Prompt Generator & Organizer`
- **Core Keywords**: `Midjourney Prompt Generator`, `AI Art Card Collector`, `Prompt Organizer`

---

## 2. Screenshot Layout Plans (5 Screenshots)

### Screenshot 1: Drag & Drop Binder Organization

- **Visual Layout**: A tablet/desktop view showing the User Library tab. A cursor is dragging a rare card from the main deck and dropping it into a custom styled collection binder. Micro-animations highlight the grid cell alignment.
- **Heading (EN)**: Organize by Drag & Drop
- **Subheading (EN)**: Intuitively catalog and group your Midjourney prompt cards.
- **Heading (JA)**: ドラッグ＆ドロップでバインダー整理
- **Subheading (JA)**: プロンプトカードを直感的に分類・コレクション。

### Screenshot 2: Holographic Glitter & Rarity Effects

- **Visual Layout**: Close-up of a "Mythic" or "Rare" tier prompt card. A shimmering glitter / holographic overlay is reflecting light as the card shifts slightly. Visual labels show Rarity values.
- **Heading (EN)**: Exquisite Rarity Tiers
- **Subheading (EN)**: Shimmering holographic effects for high-tier prompt masterpieces.
- **Heading (JA)**: 美しいレアリティとホログラフィック
- **Subheading (JA)**: 高レアリティの傑作カードに輝くグリッター演出を。

### Screenshot 3: Zero-Trust Local AI Style Analysis

- **Visual Layout**: An overlay of the "AI Analyst" panel. Clean charts showing color dominance, artistic genre classification, and prompt weight suggestions. A lock badge signifies "100% Local / Zero-Trust".
- **Heading (EN)**: Secure Local AI Analysis
- **Subheading (EN)**: Absolute privacy. Process and dissect art styles entirely on-device.
- **Heading (JA)**: 安全なローカルAIスタイル分析
- **Subheading (JA)**: 完全なプライバシー保護。端末内でスタイル傾向と配色を解析。

### Screenshot 4: Advanced Parameter & Prompt Organizer

- **Visual Layout**: Card creator view highlighting input fields for `--sref`, `--v`, and prompt segment chips. Prompt chips are neatly color-coded by theme/function.
- **Heading (EN)**: Ultimate Parameters Organizer
- **Subheading (EN)**: Manage Midjourney options, seeds, and reference codes effortlessly.
- **Heading (JA)**: 高度なパラメータ管理
- **Subheading (JA)**: `--sref` やプロンプトの各変数をカラーチップで整理。

### Screenshot 5: Direct QR & Native Sharing

- **Visual Layout**: Mobile mockup displaying the "Export Success" modal. An arrow points to the native sharing dialog activated by the Web Share API, attaching the card PNG with its embedded QR code.
- **Heading (EN)**: Share & Import Instantly
- **Subheading (EN)**: Export cards with embedded metadata. Scan to import back instantly.
- **Heading (JA)**: ワンタップで簡単共有・復元
- **Subheading (JA)**: メタデータを埋め込んだ画像を共有。QRコード読込で瞬時に復元。
